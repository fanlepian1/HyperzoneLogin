# HyperZoneLogin

实现多种登入形式的框架

## 已支持的登入方式

- 类AuthME离线登入
- 外置登入
- 正版登入

## 兼容情况

### 版本兼容情况

低于1.19.1:  
域名前加o-或者offline识别为离线玩家  
高于1.19.1:  
直接进入即可

### 启动器兼容情况

- [x] HMCL
- [x] PCL
- [x] 原版启动器
- [x] Zalith
